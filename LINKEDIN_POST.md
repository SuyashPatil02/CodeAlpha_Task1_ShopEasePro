# LinkedIn Post

Excited to share my CodeAlpha Task 1 project: **ShopEase Pro**, a complete full-stack e-commerce website built with a modern Node.js and MongoDB stack.

## Tech Stack

HTML5, CSS3, JavaScript, Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt, and Multer.

## Features

- Responsive e-commerce storefront
- 60 realistic market-ready products
- Product search, filters, cart, wishlist, checkout, and order history
- JWT authentication with secure bcrypt password hashing
- Admin dashboard for products, orders, and users
- Product image uploads using Multer
