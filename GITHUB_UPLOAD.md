# GitHub Upload Guide

```bash
cd CodeAlpha_Task1_ShopEasePro
git init
git add .
git commit -m "Add ShopEase Pro CodeAlpha Task 1 project"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/CodeAlpha_Task1_ShopEasePro.git
git push -u origin main
```

Do not commit your real `.env` file. Commit `.env.example` only.
